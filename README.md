# DSA Experiments

This project contains Data Structures and Algorithms experiments and implementations in Java.

## Deployment

This project is deployed on:
- **Vercel**: Automatic deployment from Git repository
- **Render**: Automatic deployment from Git repository

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Visit `http://localhost:3000` to view the DSA experiments.

## Content

The project contains 9 experiments covering:
1. Array Operations (1D and 2D)
2. Stack and Queue Implementation
3. Singly and Doubly Linked Lists
4. Binary Tree Traversals
5. BFS and DFS Graph Algorithms
6. Sorting Algorithms (Selection, Insertion, Merge, Quick)
7. Kruskal's Algorithm and 0/1 Knapsack
8. 8-Queens Problem using Backtracking
9. Rabin-Karp String Matching Algorithm

Each experiment includes theory, implementation, results, and conclusions.
